# DFA Atelier

Локальный редактор детерминированных автоматов.

## Как открыть

Откройте файл `index.html` в браузере.

## Опубликованный сайт

[https://alekoturtle.github.io/dfa_automaton_designer/](https://alekoturtle.github.io/dfa_automaton_designer/)

## Что умеет

- создавать вершины и перетаскивать их по холсту
- переключать форму вершины между круглой, квадратной и пятиугольной
- добавлять текстовые заметки прямо на поле автомата
- выделять несколько вершин через `Shift + Click` и рамкой на пустом месте
- перетаскивать сразу группу выделенных вершин
- создавать переходы между вершинами, включая петли
- отмечать стартовые и принимающие состояния
- проверять автомат на признаки ДКА
- загружать ранее сохраненный `TXT` обратно в редактор
- скачивать схему как `PNG`
- скачивать описание автомата как `TXT`
- приближать и отдалять холст, а также быстро подгонять вид под весь автомат

## Управление

- `V` - режим выбора
- `S` - режим добавления вершины
- `T` - режим добавления перехода
- `X` - режим добавления текста
- `Double Click` - быстро создать вершину в режиме выбора
- `Shift + Click` - добавить вершину в групповое выделение
- `Drag on Empty` - выделение рамкой
- `Shift + Wheel`, `+`, `-` - масштаб
- `Shift + 1` - показать весь автомат
- `0` - сбросить масштаб
- `Space + Drag` или `Middle Drag` - панорама
- `Delete` - удалить выделенный объект
